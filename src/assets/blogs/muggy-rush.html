game concept — “Muggy Rush”

you’re a field technician sprinting through a monsoon-season city. the catch: heat + humidity make you tire quickly. you must manage stamina and body temperature by ducking into cool zones (AC vents, mist stations, shaded alleys) while collecting parts to restart dehumidifier towers before the city fogs out.

core loop

  • move, sprint, jump, slide

  • humidity/heat index dynamically rises/falls → affects stamina drain, screen fog, footing on wet ground

  • find cool spots (reduce heat index), hydration kits (restore stamina), filters (score)

  • avoid hot, muggy pockets (steam leaks, closed rooms) that spike heat index

  • finish level before the mug gauge reaches max

systems we’ll implement (C++)

  1. UHumiditySubsystem (GameInstanceSubsystem)
    global muggy weather brain: tracks temperature, RH, computes Heat Index & Dew Point, broadcasts changes.

  2. AWeatherVolume (Actor with Box/Sphere component)
    localizes humidity/temperature (e.g., mist tunnels, AC zones).

  3. AMuggyCharacter (ACharacter)
    movement + stamina/overheat logic driven by heat index; applies camera effects and input.

  4. UMuggyPostProcessComponent (ActorComponent)
    simple link to a PostProcessVolume or material parameter collection for fog/wetness.

  5. pickup actors: hydration kit, filter cartridge.

  6. input: UE5 Enhanced Input.

note: you can keep all blueprint UIs (HUD widgets) in UMG for speed; C++ exposes values.

project setup (quick)

  1. create a new UE5 project → Games → First Person or Third Person, C++.

  2. enable plugins (Edit → Plugins):

    • Enhanced Input (on by default in new templates)

    • Niagara

  3. create folders: Source/MuggyRush and Content/MuggyRush.

  4. add a Material Parameter Collection MPC_Muggy with scalar params: FogIntensity, ScreenWetness, HeatTint.

  5. add a global PostProcessVolume (Unbound) that reads MPC values in a post-process material (can be done later; code controls params).

1) Humidity subsystem (global muggy model)

 
// HumiditySubsystem.h #pragma once #include "CoreMinimal.h" #include "Subsystems/GameInstanceSubsystem.h" #include "HumiditySubsystem.generated.h" DECLARE_DYNAMIC_MULTICAST_DELEGATE_TwoParams(FOnMuggyChanged, float, HeatIndexC, float, DewPointC); UCLASS() class MUGGYRUSH_API UHumiditySubsystem : public UGameInstanceSubsystem { GENERATED_BODY() public: UPROPERTY(BlueprintAssignable, Category="Muggy") FOnMuggyChanged OnMuggyChanged; UPROPERTY(EditAnywhere, BlueprintReadWrite, Category="Muggy") float AmbientTempC = 31.f; // base temperature UPROPERTY(EditAnywhere, BlueprintReadWrite, Category="Muggy") float AmbientRH = 78.f; // relative humidity % UPROPERTY(EditAnywhere, BlueprintReadWrite, Category="Muggy") float TimeScale = 1.f; // speed of day cycle UFUNCTION(BlueprintCallable, Category="Muggy") void SetAmbient(float InTempC, float InRH); UFUNCTION(BlueprintCallable, Category="Muggy") void ApplyLocalDelta(float DeltaTempC, float DeltaRH, float Duration); UFUNCTION(BlueprintPure, Category="Muggy") float GetHeatIndexC() const { return CachedHeatIndexC; } UFUNCTION(BlueprintPure, Category="Muggy") float GetDewPointC() const { return CachedDewPointC; } virtual void Initialize(FSubsystemCollectionBase& Collection) override; virtual void Deinitialize() override; virtual void Tick(float DeltaTime); bool IsTickable() const { return true; } private: float CachedHeatIndexC = 0.f; float CachedDewPointC = 0.f; float TempOffsetTimer = 0.f; float TempOffsetC = 0.f; float RHOffset = 0.f; void Recompute(); static float DewPointC(float T, float RH); static float HeatIndexC(float T, float RH); };
 
// HumiditySubsystem.cpp #include "HumiditySubsystem.h" #include "Engine/World.h" #include "TimerManager.h" #include "Math/UnrealMathUtility.h" void UHumiditySubsystem::Initialize(FSubsystemCollectionBase&){ Recompute(); } void UHumiditySubsystem::Deinitialize(){} void UHumiditySubsystem::SetAmbient(float InTempC, float InRH){ AmbientTempC = InTempC; AmbientRH = InRH; Recompute(); } void UHumiditySubsystem::ApplyLocalDelta(float DeltaTempC, float DeltaRH, float Duration){ TempOffsetC = DeltaTempC; RHOffset = DeltaRH; TempOffsetTimer = Duration; } void UHumiditySubsystem::Tick(float DeltaTime){ // simple slow daily swing const float Time = GetWorld() ? GetWorld()->GetTimeSeconds() * TimeScale : 0.f; const float DaySine = FMath::Sin(Time * 2.f * PI / 120.f); // 2min loop demo const float TempDaily = AmbientTempC + 2.5f * DaySine; const float RHDaily = AmbientRH + 5.f * FMath::Max(0.f, DaySine); if (TempOffsetTimer > 0.f){ TempOffsetTimer -= DeltaTime; if (TempOffsetTimer <= 0.f){ TempOffsetC = 0.f; RHOffset = 0.f; } } const float T = TempDaily + TempOffsetC; const float RH = FMath::Clamp(RHDaily + RHOffset, 0.f, 100.f); const float NewDP = DewPointC(T, RH); const float NewHI = HeatIndexC(T, RH); if (!FMath::IsNearlyEqual(NewDP, CachedDewPointC, 0.01f) || !FMath::IsNearlyEqual(NewHI, CachedHeatIndexC, 0.01f)) { CachedDewPointC = NewDP; CachedHeatIndexC = NewHI; OnMuggyChanged.Broadcast(CachedHeatIndexC, CachedDewPointC); } } float UHumiditySubsystem::DewPointC(float T, float RH){ const float a=17.62f, b=243.12f; const float gamma = (a*T)/(b+T) + FMath::Loge(RH/100.f); return (b*gamma)/(a-gamma); } float UHumiditySubsystem::HeatIndexC(float T, float RH){ const float TF = T*9.f/5.f + 32.f; const float R = RH; float HI = -42.379f + 2.04901523f*TF + 10.14333127f*R - 0.22475541f*TF*R - 0.00683783f*TF*TF - 0.05481717f*R*R + 0.00122874f*TF*TF*R + 0.00085282f*TF*R*R - 0.00000199f*TF*TF*R*R; if (TF < 80.f){ const float w = FMath::Clamp((TF-70.f)/10.f, 0.f, 1.f); HI = (1.f-w)*TF + w*HI; } return (HI-32.f)*5.f/9.f; }

this gives you a tickable, global humidity model with a broadcast you can hook to VFX and gameplay.

2) local weather volumes (cool spots / steam pockets)

// WeatherVolume.cpp #include "WeatherVolume.h" #include "Components/BoxComponent.h" #include "HumiditySubsystem.h" #include "Kismet/GameplayStatics.h" AWeatherVolume::AWeatherVolume(){ Box = CreateDefaultSubobject<UBoxComponent>(TEXT("Box")); RootComponent = Box; Box->SetCollisionProfileName(TEXT("OverlapAll")); Box->OnComponentBeginOverlap.AddDynamic(this, &AWeatherVolume::OnBeginOverlap); Box->OnComponentEndOverlap.AddDynamic(this, &AWeatherVolume::OnEndOverlap); } void AWeatherVolume::OnBeginOverlap(AActor*, AActor* Other){ if (!Other) return; if (UWorld* W = GetWorld()){ if (auto* HS = W->GetGameInstance()->GetSubsystem<UHumiditySubsystem>()){ HS->ApplyLocalDelta(DeltaTempC, DeltaRH, EffectDuration); } } } void AWeatherVolume::OnEndOverlap(AActor*, AActor* Other){ if (!Other) return; if (UWorld* W = GetWorld()){ if (auto* HS = W->GetGameInstance()->GetSubsystem<UHumiditySubsystem>()){ HS->ApplyLocalDelta(0.f, 0.f, 0.1f); } } }

drop these in the level to make AC zones (negative deltas) or steam leaks (positive deltas).

3) post-process driver (fog/wetness feedback)

 
// MuggyPostProcessComponent.h #pragma once #include "CoreMinimal.h" #include "Components/ActorComponent.h" #include "MuggyPostProcessComponent.generated.h" UCLASS(ClassGroup=(Custom), meta=(BlueprintSpawnableComponent)) class MUGGYRUSH_API UMuggyPostProcessComponent : public UActorComponent { GENERATED_BODY() public: virtual void BeginPlay() override; UPROPERTY(EditAnywhere, Category="Muggy") FName FogParam = "FogIntensity"; UPROPERTY(EditAnywhere, Category="Muggy") FName WetParam = "ScreenWetness"; UPROPERTY(EditAnywhere, Category="Muggy") FName HeatParam = "HeatTint"; private: class UMaterialParameterCollection* MPC; class UMaterialParameterCollectionInstance* MPCI; UFUNCTION() void OnMuggyChanged(float HeatIndexC, float DewPointC); };
 
// MuggyPostProcessComponent.cpp #include "MuggyPostProcessComponent.h" #include "HumiditySubsystem.h" #include "Materials/MaterialParameterCollection.h" #include "Materials/MaterialParameterCollectionInstance.h" #include "Kismet/GameplayStatics.h" void UMuggyPostProcessComponent::BeginPlay(){ Super::BeginPlay(); MPC = LoadObject<UMaterialParameterCollection>(nullptr, TEXT("/Game/MuggyRush/MPC_Muggy.MPC_Muggy")); if (MPC){ MPCI = GetWorld()->GetParameterCollectionInstance(MPC); } if (auto* HS = GetWorld()->GetGameInstance()->GetSubsystem<UHumiditySubsystem>()){ HS->OnMuggyChanged.AddDynamic(this, &UMuggyPostProcessComponent::OnMuggyChanged); } } void UMuggyPostProcessComponent::OnMuggyChanged(float HeatIndexC, float DewPointC){ if (!MPCI) return; const float Fog = FMath::GetMappedRangeValueClamped(FVector2D(30.f, 45.f), FVector2D(0.1f, 1.0f), HeatIndexC); const float Wet = FMath::GetMappedRangeValueClamped(FVector2D(22.f, 27.f), FVector2D(0.0f, 1.0f), DewPointC); const float HeatTint = FMath::GetMappedRangeValueClamped(FVector2D(32.f, 44.f), FVector2D(0.0f, 1.0f), HeatIndexC); MPCI->SetScalarParameterValue(FogParam, Fog); MPCI->SetScalarParameterValue(WetParam, Wet); MPCI->SetScalarParameterValue(HeatParam, HeatTint); }

attach this component to the player (or a manager actor) to drive fog/wetness in real time.

4) player character with stamina & overheating

// MuggyCharacter.cpp #include "MuggyCharacter.h" #include "Camera/CameraComponent.h" #include "GameFramework/SpringArmComponent.h" #include "EnhancedInputSubsystems.h" #include "EnhancedInputComponent.h" #include "HumiditySubsystem.h" #include "MuggyPostProcessComponent.h" #include "GameFramework/CharacterMovementComponent.h" #include "Kismet/GameplayStatics.h" AMuggyCharacter::AMuggyCharacter(){ PrimaryActorTick.bCanEverTick = true; SpringArm = CreateDefaultSubobject<USpringArmComponent>(TEXT("SpringArm")); SpringArm->SetupAttachment(RootComponent); SpringArm->TargetArmLength = 0.f; // first person? set 0 or small for third person set 300 Camera = CreateDefaultSubobject<UCameraComponent>(TEXT("Camera")); Camera->SetupAttachment(SpringArm); MuggyFX = CreateDefaultSubobject<UMuggyPostProcessComponent>(TEXT("MuggyFX")); GetCharacterMovement()->MaxWalkSpeed = 450.f; } void AMuggyCharacter::BeginPlay(){ Super::BeginPlay(); if (APlayerController* PC = Cast<APlayerController>(GetController())){ if (ULocalPlayer* LP = PC->GetLocalPlayer()){ if (auto* Sub = LP->GetSubsystem<UEnhancedInputLocalPlayerSubsystem>()){ Sub->AddMappingContext(IMC, 0); } } } } void AMuggyCharacter::SetupPlayerInputComponent(UInputComponent* PlayerInputComponent){ if (UEnhancedInputComponent* EIC = Cast<UEnhancedInputComponent>(PlayerInputComponent)){ EIC->BindAction(IA_Move, ETriggerEvent::Triggered, this, &AMuggyCharacter::Move); EIC->BindAction(IA_Look, ETriggerEvent::Triggered, this, &AMuggyCharacter::Look); EIC->BindAction(IA_Sprint, ETriggerEvent::Started, this, &AMuggyCharacter::SprintPressed); EIC->BindAction(IA_Sprint, ETriggerEvent::Completed, this, &AMuggyCharacter::SprintReleased); EIC->BindAction(IA_Jump, ETriggerEvent::Started, this, &ACharacter::Jump); } } void AMuggyCharacter::Move(const FInputActionValue& Value){ FVector2D Axis = Value.Get<FVector2D>(); MoveInput = Axis; if (Controller){ AddMovementInput(GetActorForwardVector(), Axis.Y); AddMovementInput(GetActorRightVector(), Axis.X); } } void AMuggyCharacter::Look(const FInputActionValue& Value){ FVector2D Axis = Value.Get<FVector2D>(); AddControllerYawInput(Axis.X); AddControllerPitchInput(Axis.Y); } void AMuggyCharacter::SprintPressed(const FInputActionValue&){ bSprinting = true; RefreshMovementSpeed(); } void AMuggyCharacter::SprintReleased(const FInputActionValue&){ bSprinting = false; RefreshMovementSpeed(); } void AMuggyCharacter::RefreshMovementSpeed(){ float Base = 450.f; float Speed = bSprinting ? 700.f : Base; // heat penalty if (UWorld* W = GetWorld()){ if (auto* HS = W->GetGameInstance()->GetSubsystem<UHumiditySubsystem>()){ const float HI = HS->GetHeatIndexC(); // 30..45 typical const float Penalty = FMath::GetMappedRangeValueClamped(FVector2D(30.f, 45.f), FVector2D(0.f, 200.f), HI); Speed = FMath::Max(250.f, Speed - Penalty*0.5f); } } GetCharacterMovement()->MaxWalkSpeed = Speed; } void AMuggyCharacter::Tick(float DeltaSeconds){ Super::Tick(DeltaSeconds); float HI = 32.f; if (auto* HS = GetWorld()->GetGameInstance()->GetSubsystem<UHumiditySubsystem>()){ HI = HS->GetHeatIndexC(); } // stamina dynamics const bool bIsMovingFast = bSprinting && (MoveInput.SizeSquared() > 0.1f); const float Drain = bIsMovingFast ? (BaseStaminaDrain + HeatSensitivity * FMath::Max(0.f, HI-30.f)) : 0.f; const float Regen = (!bIsMovingFast && HI < 35.f) ? 10.f : 4.f; Stamina = FMath::Clamp(Stamina + ( -Drain + Regen) * DeltaSeconds, 0.f, MaxStamina); BodyHeat = FMath::Clamp(BodyHeat + (bIsMovingFast ? (0.6f + 0.05f*(HI-30.f)) : -0.8f) * DeltaSeconds, 0.f, 100.f); if (Stamina <= 1.f || BodyHeat >= 95.f){ bSprinting = false; // auto cancel sprint } RefreshMovementSpeed(); }

add a FilterPickup for scoring the dehumidifier objective in the same pattern.

6) win/lose conditions (mug gauge)

  • win: collect N filters and reach the dehumidifier tower.

  • lose: Mug Gauge hits max (e.g., HI averaged over 20s > 42°C) or timer expires.

Add a simple GameMode or Level Blueprint:

  • listen to HumiditySubsystem.OnMuggyChanged

  • maintain a rolling average of heat index

  • if threshold exceeded → trigger fog burst + fade to black.

7) visuals & “latest tech” sprinkles

  • Lumen: default global illumination—use wet asphalt and metallic props; looks great under overcast HDRIs.

  • Nanite: enable on dense meshes; helps performance.

  • Niagara:

    • soft mist emitters in cool tunnels

    • subtle heat shimmer in hot alleys (screen-aligned distortion)

  • Material functions: WorldAlignedWetness driven by ScreenWetness MPC param for a universal “wetness” mask.

  • Chaos: slippery physics—reduce friction on wet floor Physics Materials (character will slide a bit when sprinting with low stamina).

  3. set HumiditySubsystem.AmbientTempC = 32, AmbientRH = 80.

level design tips (fast)

  • use a foggy HDRI sky + slightly desaturated LUT.

  • put cool refuges every ~15–25s of sprint time.

  • path choices: short/hot alley vs long/cool arcade.

  • readable VFX: blue-white mist for cool, orange shimmer for hot.
