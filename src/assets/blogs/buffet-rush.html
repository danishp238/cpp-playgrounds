üéØ Game Concept

"Buffet Rush" is a fun, text-based or SFML-powered graphical C++ game where the player acts as a customer in a buffet. The goal is to build the perfect meal tray under time and nutritional constraints—while also managing cost, calories, and customer satisfaction. The game introduces logic building, simple AI, and real-life problem solving in a kitchen-inspired setting.

üß± Core Game Features

  • Choose items from a buffet counter.

  • Each item has:

    • Calories

    • Cost

    • Category (Starter, Main, Dessert, Drink)

    • Nutritional Value

  • User wins if they build a balanced tray (one from each category) under the calorie and budget limit.

üéÆ SFML Sample Snippet: Bars + Logic

Here’s a simplified implementation:

cpp
#include <SFML/Graphics.hpp> #include <iostream> #include <vector> const int calorieLimit = 900; int currentCalories = 0; int score = 0; sf::RectangleShape createBar(sf::Vector2f position, sf::Color color, float percentage) { sf::RectangleShape bar(sf::Vector2f(200 * percentage, 20)); bar.setPosition(position); bar.setFillColor(color); return bar; } int main() { sf::RenderWindow window(sf::VideoMode(600, 400), "Buffet Rush - Score & Nutrition"); sf::Font font; if (!font.loadFromFile("arial.ttf")) return -1; sf::Text title("Buffet Rush!", font, 30); title.setPosition(180, 10); sf::Text caloriesText; caloriesText.setFont(font); caloriesText.setCharacterSize(16); caloriesText.setPosition(50, 70); sf::Text scoreText; scoreText.setFont(font); scoreText.setCharacterSize(16); scoreText.setPosition(50, 140); // Simulated calorie increase std::vector<int> choices = {100, 250, 300, 180}; // Example selections for (int cal : choices) { currentCalories += cal; score += (cal < 300 ? 10 : 5); // Healthy = more score } while (window.isOpen()) { sf::Event event; while (window.pollEvent(event)) { if (event.type == sf::Event::Closed) window.close(); } float caloriePercent = static_cast<float>(currentCalories) / calorieLimit; if (caloriePercent > 1.0f) caloriePercent = 1.0f; float scorePercent = static_cast<float>(score) / 50.0f; // Assume 50 is max if (scorePercent > 1.0f) scorePercent = 1.0f; caloriesText.setString("Calories: " + std::to_string(currentCalories) + " / 900"); scoreText.setString("Score: " + std::to_string(score)); window.clear(sf::Color(240, 240, 240)); window.draw(title); window.draw(caloriesText); window.draw(createBar(sf::Vector2f(50, 100), sf::Color::Red, caloriePercent)); window.draw(scoreText); window.draw(createBar(sf::Vector2f(50, 170), sf::Color::Green, scorePercent)); window.display(); } return 0; }

üß† Explanation

  • We simulate item choices by adding calories and scores.

  • Bars grow in width proportionally using percentages.

  • The red bar shows nutrition intake, green bar shows score progress.
