Rainfall Rescue – Building a Fun SFML C++ Game

Introduction – Bringing the Rain to Life in Code

Have you ever wished you could control the rain? In this exciting C++ project, we’re doing just that – building Rainfall Rescue, a 2D arcade-style game where players catch pure rain and avoid acid drops.

This game combines:

  • Fun gameplay

  • Real-world environmental concepts

  • The power of SFML (Simple and Fast Multimedia Library)

In Rainfall Rescue, you’ll move a bucket left and right to collect clean rainwater. Every drop you catch boosts your score and helps plants grow, while acid rain harms them. Add realistic rain sounds, splashes, and simple particle effects, and you’ve got a truly immersive coding project.

Section 1 – Setting Up SFML on Windows

You’ll need:

  • Windows PC with a C++ compiler (MinGW or Visual Studio recommended)

  • SFML 2.5+ from sfml-dev.org

  • A project folder containing:

    • Rainfall_SFML_Game.cpp

    • An assets folder with your font (Roboto-Regular.ttf) and sound files

Compile with MinGW:

bash
g++ -std=c++17 -O2 -I"C:\SFML\include" Rainfall_SFML_Game.cpp -o Rainfall.exe -L"C:\SFML\lib" -lsfml-graphics -lsfml-window -lsfml-system -lsfml-audio

Section 2 – Game Features

  • Dynamic raindrops: clean drops to catch, acid drops to avoid

  • Player-controlled bucket with smooth movement

  • Real-time scoring and plant health indicators

  • Sound effects for rain, splashes, and acid hissing

  • Particle bursts for water splashes

  • Simple, clean visuals without external sprites

Section 3 – The Code, Explained Step-by-Step

#include <SFML/Graphics.hpp>
#include <vector>
#include <random>
#include <string>
#include <sstream>

struct Raindrop {
sf::CircleShape shape;
float speed;
bool acid;
};

int main() {
const unsigned int WINDOW_W = 1024;
const unsigned int WINDOW_H = 720;
sf::RenderWindow window(sf::VideoMode(WINDOW_W, WINDOW_H), "Rainfall Eco-Proto");
window.setFramerateLimit(60);

std::random_device rd;
std::mt19937 rng(rd());
std::uniform_real_distribution<float> xDist(20.f, WINDOW_W - 20.f);
std::uniform_real_distribution<float> speedDist(140.f, 360.f);
std::uniform_real_distribution<float> acidChance(0.f, 1.f);

sf::RectangleShape bucket(sf::Vector2f(120.f, 24.f));
bucket.setOrigin(bucket.getSize().x/2.f, bucket.getSize().y/2.f);
bucket.setPosition(WINDOW_W/2.f, WINDOW_H - 60.f);
bucket.setFillColor(sf::Color(100, 149, 237));

sf::RectangleShape ground(sf::Vector2f((float)WINDOW_W, 100.f));
ground.setPosition(0.f, WINDOW_H - 50.f);
ground.setFillColor(sf::Color(50, 160, 60));

float plantHealth = 0.6f;
sf::CircleShape plant(28.f);
plant.setOrigin(plant.getRadius(), plant.getRadius());
plant.setPosition(100.f, WINDOW_H - 90.f);

sf::Font font;
bool fontLoaded = font.loadFromFile("assets/Roboto-Regular.ttf");

sf::Text hudText;
if (fontLoaded) {
hudText.setFont(font);
hudText.setCharacterSize(18);
hudText.setFillColor(sf::Color::White);
hudText.setPosition(10.f, 10.f);
}

std::vector<Raindrop> drops;
float spawnTimer = 0.f;
float spawnInterval = 0.15f;
int score = 0;
float soilMoisture = 0.3f;
float gameTime = 60.f;

sf::Clock clock;

auto spawnDrop = [&](float overrideX = -1.f) {
Raindrop d;
d.shape = sf::CircleShape(8.f);
float x = (overrideX < 0.f) ? xDist(rng) : overrideX;
d.shape.setPosition(x, -10.f);
d.speed = speedDist(rng);
d.acid = (acidChance(rng) < 0.12f);
d.shape.setFillColor(d.acid ? sf::Color(170, 0, 255) : sf::Color(120, 180, 255));
drops.push_back(std::move(d));
};

for (int i=0;i<6;i++) spawnDrop();

while (window.isOpen()) {
float dt = clock.restart().asSeconds();
gameTime -= dt;

sf::Event ev;
while (window.pollEvent(ev)) {
if (ev.type == sf::Event::Closed) window.close();
}

if (sf::Keyboard::isKeyPressed(sf::Keyboard::Left) || sf::Keyboard::isKeyPressed(sf::Keyboard::A))
bucket.move(-420.f * dt, 0.f);
if (sf::Keyboard::isKeyPressed(sf::Keyboard::Right) || sf::Keyboard::isKeyPressed(sf::Keyboard::D))
bucket.move(420.f * dt, 0.f);

if (bucket.getPosition().x < bucket.getSize().x/2.f) bucket.setPosition(bucket.getSize().x/2.f, bucket.getPosition().y);
if (bucket.getPosition().x > WINDOW_W - bucket.getSize().x/2.f) bucket.setPosition(WINDOW_W - bucket.getSize().x/2.f, bucket.getPosition().y);

spawnTimer += dt;
if (spawnTimer >= spawnInterval) {
spawnTimer = 0.f;
spawnDrop();
if (acidChance(rng) < 0.08f) {
float clusterX = xDist(rng);
for (int i=0;i<4;i++) spawnDrop(clusterX + (i-2)*12.f);
}
}

for (size_t i=0;i<drops.size();) {
drops[i].shape.move(0.f, drops[i].speed * dt);
if (drops[i].shape.getGlobalBounds().intersects(bucket.getGlobalBounds())) {
if (drops[i].acid) {
score -= 5;
soilMoisture = std::max(0.f, soilMoisture - 0.06f);
plantHealth = std::max(0.f, plantHealth - 0.06f);
} else {
score += 10;
soilMoisture = std::min(1.f, soilMoisture + 0.05f);
plantHealth = std::min(1.f, plantHealth + 0.03f);
}
drops.erase(drops.begin() + i);
continue;
}
if (drops[i].shape.getPosition().y > WINDOW_H - 40.f) {
if (drops[i].acid) {
plantHealth = std::max(0.f, plantHealth - 0.03f);
} else {
soilMoisture = std::min(1.f, soilMoisture + 0.02f);
}
drops.erase(drops.begin() + i);
continue;
}
++i;
}

if (soilMoisture < 0.4f) plantHealth = std::max(0.f, plantHealth - 0.01f * dt);
else plantHealth = std::min(1.f, plantHealth + 0.02f * dt);

soilMoisture = std::max(0.f, soilMoisture - 0.01f * dt);

float plantScale = 0.6f + plantHealth * 1.2f;
plant.setRadius(28.f * plantScale);
plant.setOrigin(plant.getRadius(), plant.getRadius());
sf::Uint8 g = (sf::Uint8)(100 + plantHealth * 155);
sf::Uint8 r = (sf::Uint8)(60 + (1.f-plantHealth) * 120);
plant.setFillColor(sf::Color(r, g, 40));

window.clear(sf::Color(20, 24, 40));
for (auto &d : drops) window.draw(d.shape);
window.draw(ground);
window.draw(plant);
window.draw(bucket);

if (fontLoaded) {
std::stringstream s;
s << "Score: " << score << " Time: " << (int)gameTime << "s";
s << " Soil: " << (int)(soilMoisture*100) << "% Plant: " << (int)(plantHealth*100) << "%";
hudText.setString(s.str());
window.draw(hudText);
}

window.display();
}
}

1. Including Libraries

We include SFML modules for graphics, window control, and audio. This provides shapes, text rendering, sound playback, and more.

2. Defining Game Objects

  • Bucket → sf::RectangleShape

  • Raindrops → sf::CircleShape

  • Particles → stored in std::vector for splashes

3. Handling Input

We check for keyboard events to move the bucket left or right. Both arrow keys and A/D keys are supported for comfort.

4. Spawning Rain

  • Timer-controlled raindrop creation

  • Random positions and speeds

  • Probability to decide if a drop is clean or acid

5. Collision Detection

If a drop hits the bucket:

  • Clean drop → increase score

  • Acid drop → reduce plant health

6. Adding Sound

We load .ogg or .wav into sf::SoundBuffer and attach to sf::Sound objects.

  • Rain background loops

  • Splash sound for catching drops

  • Acid hiss for acid rain impact

7. Particle Effects

When drops hit, small fading circles simulate splashes.

8. Drawing the Frame

Every frame:

  • Clear the window

  • Draw bucket, raindrops, particles, HUD

Fun Facts About Rain

  • The heaviest raindrop recorded was 8.8 mm in diameter.

  • Acid rain can damage plants, fish, and even buildings.

  • Rainbows form when sunlight refracts through raindrops – not acid drops!
